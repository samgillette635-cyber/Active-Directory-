# Active-Directory-
# Enterprise Active Directory Lab

## Overview
Built a Windows enterprise environment using Windows Server 2022 and Windows 11 to simulate a real corporate Active Directory network.

## Objectives
- Deploy Active Directory Domain Services
- Create Organizational Units (OUs)
- Configure users and security groups
- Apply the Principle of Least Privilege
- Automate drive mapping using Group Policy

## Environment
- Windows Server 2022
- Windows 11
- VMware Workstation
- Active Directory
- Group Policy
- DNS
- File Server

## What I Configured
- Domain Controller
- File Server
- Sales, HR, and IT Organizational Units
- Department security groups
- Shared network drives
- NTFS permissions
- Group Policy Objects
- Item-Level Targeting
- Least privilege access
